
O banco de dados foi modelado para gerenciar uma oficina, organizando informações sobre clientes, veículos, funcionários, servicos, ordens de serviço (OS) e pagamentos. A lógica segue os seguintes princípios:

Clientes, Veículos, Serviços, Ordens de Serviço (OS), Funcionários e Pagamentos.

Os relacionamentos garantem integridade referencial, permitindo consultas complexas como listagem de serviços realizados por funcionario, veículos atendidos e histórico de OS de cada cliente.
